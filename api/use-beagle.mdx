---
title: 'useBeagle'
description: 'Hook for accessing Beagle context and inspector controls'
---

The `useBeagle` hook provides access to the Beagle context, allowing you to interact with logs, control the inspector, and access copy functionality.

## Return Value

The hook returns an object with the following properties:

<ResponseField name="logs" type="Array<BeagleLog>">
  Array of all captured logs. Each log contains id, time, message, and level properties.
</ResponseField>

<ResponseField name="addLog" type="(log: BeagleLog) => void">
  Function to manually add a log to the inspector. Typically used internally by plugins, but can be called directly if needed.
</ResponseField>

<ResponseField name="showInspector" type="() => void">
  Opens the Beagle inspector modal, displaying all captured logs.
</ResponseField>

<ResponseField name="closeInspector" type="() => void">
  Closes the Beagle inspector modal.
</ResponseField>

<ResponseField name="copy" type="((value: string) => Promise<void>) | undefined">
  Copy function provided to `BeagleProvider`. May be `undefined` if no copy function was configured.
</ResponseField>

## Usage

```typescript
import { useBeagle } from 'react-native-beagle';
import { View, Button, Text } from 'react-native';

function DebugPanel() {
  const { logs, showInspector, copy } = useBeagle();

  return (
    <View>
      <Text>Total Logs: {logs.length}</Text>
      <Button
        title="Open Inspector"
        onPress={showInspector}
      />
    </View>
  );
}
```

## Opening the Inspector

The most common use case is to provide a UI element that opens the inspector:

```typescript
import { useBeagle } from 'react-native-beagle';
import { TouchableOpacity, Text } from 'react-native';

function FloatingDebugButton() {
  const { showInspector, logs } = useBeagle();

  return (
    <TouchableOpacity
      onPress={showInspector}
      style={{
        position: 'absolute',
        bottom: 20,
        right: 20,
        backgroundColor: '#007bff',
        padding: 10,
        borderRadius: 20,
      }}
    >
      <Text style={{ color: 'white' }}>
        {logs.length} Logs
      </Text>
    </TouchableOpacity>
  );
}
```

## Accessing Logs

You can access and filter logs directly:

```typescript
import { useBeagle } from 'react-native-beagle';

function ErrorCounter() {
  const { logs } = useBeagle();
  const errorCount = logs.filter(log => log.level === 'error').length;

  return <Text>Errors: {errorCount}</Text>;
}
```

## Manual Logging

While `Beagle.log()` is the recommended approach, you can also use `addLog` directly:

```typescript
import { useBeagle } from 'react-native-beagle';
import { BeagleLog } from 'react-native-beagle';

function CustomLogger() {
  const { addLog } = useBeagle();

  const logCustomEvent = () => {
    const log = new BeagleLog('Custom event occurred', 'info');
    addLog(log);
  };

  return <Button title="Log Event" onPress={logCustomEvent} />;
}
```

## Type Definition

```typescript
interface BeagleContextValue {
  logs: Array<BeagleLog>;
  addLog: (log: BeagleLog) => void;
  showInspector: () => void;
  closeInspector: () => void;
  copy?: (value: string) => Promise<void>;
}
```
